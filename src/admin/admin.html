How to customize

1. Change services, prices, materials

Go to Services & Pricing in the sidebar. Each service has a pricing type — pick whichever fits:

  • Flat — one fixed price (e.g., $99 for window cleaning)
  • Per unit — price × quantity (e.g., $0.25 per sq ft)
  • Tiered — price based on size range (e.g., "up to 1000 sqft = $275")
  • Matrix — price depends on multiple factors combined (e.g., floors × material × size)

2. Preview your changes

Hit Save & Preview in the top bar. Your edits are stored in the browser so you can iterate. When ready, go to Export and download the JSON.

3. Ship to your site

Replace src/core/config.default.json in the repo with the exported file, commit, push. That's it.

Packages via URL

Packages are pre-selected bundles. Linking to form.html?package=p1 auto-checks the services in package p1. Useful for marketing emails and landing page CTAs.
